# 🗾 埼玉県公共施設ダッシュボード

![thumbnail](thumbnail.png)

## プロジェクト概要
Tableau Publicを用いて、埼玉県内の公共施設分布とカテゴリ傾向を可視化しました。
以下を目的としています：

- 施設の種類ごとの分布を地図で把握
- ヒートマップにより密集地域を一目で確認
- カテゴリ別件数を棒グラフで比較

---

## 🔍 使用ツール
- Tableau Public（ダッシュボード作成）
- Python（前処理：pandas）
- GitHub（ポートフォリオ管理）

---

## 🗺️ ダッシュボード構成
| セクション | 内容 |
|------------|------|
| 🧭 マップ / ヒートマップ切替 | パラメータで切替可能（🗺️ / 🔥）|
| 📊 施設カテゴリ別件数 | 棒グラフ表示（横向き）|
